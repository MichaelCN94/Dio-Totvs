# Primeiros passos

## Criando e clonando Repositorios

### Git init

Inicia um no diretorio atual.

    - Criar pasta para iniciar o git $ mkdir "nome do diretorio"
    - Iniciar o git - $ git init

### Clonagem Git

Clona um repositorio na sua maquina

    - git clone "link" "nome da pasta"

### Conectando repositorio remoto

Para conectar o repositorio remoto:

    - $ git remote add origin "url"
